## ğŸš€ Getting Started

### Prerequisites

- Node.js (LTS version recommended)

### Installation

1. Clone the repository
```bash
git clone https://github.com/jorioe/local-news-pulse.git
cd local-news-pulse
```

2. Install dependencies
```bash
npm install
# or

```

3. Start the development server
```bash
npm run dev
# or

```
